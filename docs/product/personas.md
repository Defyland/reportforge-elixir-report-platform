# Personas

## Finance Operations User

Needs recurring exports for cash position, ledger reconciliation, and invoice
audit workflows. Values predictable completion, downloadable artifacts, and the
ability to give support teams a report ID when something goes wrong.

## Data Operations User

Uses exports to support reconciliation and internal audit requests. Values
stable templates, clear filters, immutable event history, and artifacts with
checksums.

## Platform Engineer

Owns reliability of the reporting platform. Values idempotency, durable jobs,
bounded storage responsibilities, health checks, metrics, traces, runbooks, and
clear deployment constraints.

## Support Or Reliability Engineer

Investigates delayed, failed, duplicate, or exposed reports. Values lifecycle
events, audit logs, normalized authorization behavior, request IDs, correlation
IDs, and runbooks for incident response.
